Character 클래스

```java
// 대문자 여부
Character.isUpperCase(ch)

// 소문자 여부
Character.isLowerCase(ch)

// 문자 여부
Character.isLetter(ch)

// 숫자 여부
Character.isDigit(ch)

// 공백 문자 여부
Character.isWhitespace(ch)

// 문자 또는 숫자 여부
Character.isLetterOrDigit(ch)

// 대문자로 변환
Character.toUpperCase(ch)

// 소문자로 변환
Character.toLowerCase(ch)
```
